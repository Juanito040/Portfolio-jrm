# Portafolio personal

Sitio web de portafolio construido con React y Vite.

## Tecnologías

- React 19 + Vite
- Framer Motion
- SCSS Modules
- EmailJS (formulario de contacto)

## Cómo correrlo

```bash
git clone https://github.com/tu-usuario/tu-repo.git
cd tu-repo
npm install
npm run dev
```

## Scripts

```bash
npm run dev      # desarrollo
npm run build    # producción
npm run preview  # preview del build
```
